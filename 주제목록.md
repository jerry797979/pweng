## 예정 — 시험·자격증 (점수가 급한 사람 공략)

각 페이지는 시험 정보 + '암기가 아닌 매일 발화 훈련'이라는 관점으로 연결한다. 시험 제도·채점 기준 같은 사실 정보는 일반적으로 알려진 수준만 쓰고 구체적 수치가 불확실하면 단정하지 않는다. 기존 jobseeker-english, phone-interview-english와 겹치지 않게 '시험 자체'에만 집중한다.

- opic-il-to-ih : 오픽 IL에서 IH로 — 등급이 안 오르는 사람의 공통점
- opic-topics : 오픽 서베이 전략과 단골 주제 — 답변을 준비가 아닌 훈련으로
- ielts-speaking-guide : 아이엘츠 스피킹 밴드 올리기 — 파트별 채점 기준과 훈련법
- ielts-speaking-part2 : 아이엘츠 스피킹 파트2 — 2분 말하기가 무너지는 이유
- opic-study-period : 오픽 몇 주 준비하면 될까 — 현재 레벨별 현실적인 기간
- speaking-test-nervous : 스피킹 시험 긴장 — 실전에서 얼어붙지 않는 훈련법
- exam-template-limit : 스피킹 시험 템플릿 암기가 통하지 않는 이유
- opic-retake : 오픽 재응시 전략 — 같은 공부를 반복하면 같은 등급이 나온다
- promotion-english-score : 승진·이직에 필요한 영어 스피킹 점수 — 기업별 기준
- toeic-vs-speaking : 토익 900인데 말을 못하는 이유 — 점수와 회화의 간극
- business-english-cert : 비즈니스 영어 자격 비교 — 오픽·토스·BULATS 뭐가 인정받나
- exam-with-daily-practice : 스피킹 시험, 학원 없이 매일 훈련으로 준비하는 법
