## 작업 지시

1. 아래 목록의 키워드마다 랜딩페이지를 하나씩 만든다.
2. 폴더 이름은 각 항목의 **폴더명을 그대로** 쓴다. 새로 짓거나 바꾸지 말 것. (주소가 이미 CRM에 등록되어 있음)
3. 각 폴더 안에 index.html 을 만든다.
4. 디자인·구성·문체는 프로젝트의 CLAUDE.md 규칙을 그대로 따른다.
5. Article + FAQPage JSON-LD 스키마를 반드시 넣고, 넣은 뒤 유효한지 확인한다.
6. 브랜드 관련 사실은 CLAUDE.md에 확정된 것만 쓴다. 수치·이력·후기를 지어내지 말 것.
7. 각 페이지의 title, meta description 을 키워드에 맞게 쓰고 서로 겹치지 않게 한다.
8. 기존 관련 페이지가 있으면 서로 연결되도록 안쪽 링크를 넣는다.
9. 아래 '기획 자료'는 방향 참고용이다. 그대로 옮겨 쓰지 말고 페이지 글로 다시 쓴다.
